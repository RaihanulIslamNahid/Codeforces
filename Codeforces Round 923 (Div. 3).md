# Codeforces Round 923 (Div. 3)

  ## A - Make it White
  Find the first and last B, that's the length.

  ## B - Following the String
  Whenever 0 comes up, give a new character. Otherwise, for any x, pick any of the unused previous (x - 1) and give that character.

  ## C - Choose the Different Ones!
  Check three conditions. All elements are present in at least one of these. Elements present in only A are less than k / 2 and the same for elements only in B.

  ## D - Find the Different Ones!
  For every value we can store indices of when it comes for the first time in any contiguous occurrences. As in if from x to x + 4, this value is present, we store x.
  For every query we find the index for A[r], if there is such after l, then we know the answer, otherwise it means all are the same.
  ## E - Klever Permutation
  One possible way can be this, for all odd indices 1, 1 + k, 1 + 2k. Start from 1 and keep increasing. For even indices, start from n and keep decreasing.
  This way we first increase the sum and then decrease it, hence diff is never greater than 1.
